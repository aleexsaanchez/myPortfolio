Featured project

Loudflare Edge Security Lab

Visit live site

A hardened serverless environment moved to the global edge.

I built and hardened a security-focused environment to reduce exposure, control access, and make the perimeter smarter. The goal was not just to make something that works, but something that reflects how I think about modern blue-team defense.

  • Zero Trust access with GitHub OAuth and OTP
  • WAF hardening for SQLi, XSS, and reconnaissance probes
  • Cloudflare Workers for security headers and information masking
  • Serverless-first approach to reduce public origin exposure
